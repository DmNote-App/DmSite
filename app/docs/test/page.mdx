# 테스트 문서

## 소개

이것은 테스트용 마크다운 문서입니다. 다양한 마크다운 요소들을 포함하고 있습니다.

## 텍스트 서식

일반 텍스트입니다. **굵은 글씨**와 _기울임꼴_, 그리고 ~~취소선~~도 사용할 수 있습니다.

`인라인 코드`도 작성할 수 있습니다.

## 코드 블록

```javascript
function hello() {
  console.log("Hello, World!");
  return true;
}
```

```python
def greet(name):
    print(f"안녕하세요, {name}님!")
    return True
```
