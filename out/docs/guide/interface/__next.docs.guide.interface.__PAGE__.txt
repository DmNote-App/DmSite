# 인터페이스 안내

DM Note는 두 개의 독립적인 창으로 구성됩니다.

## 메인 창

메인 창은 프로그램의 모든 설정을 관리하는 곳입니다.

### 상단 탭

- **Keys**: 키 매핑과 배치를 관리합니다.
- **Settings**: 프로그램 설정을 변경합니다.

### Keys 탭

키 매핑 탭에서는 다음 기능을 사용할 수 있습니다:

| 기능         | 설명                                                           |
| ------------ | -------------------------------------------------------------- |
| 키 모드 선택 | 4key, 5key, 6key, 8key 등 다양한 키 모드를 선택할 수 있습니다. |
| 키 추가/삭제 | 그리드에서 우클릭하여 키를 추가하거나 삭제합니다.              |
| 키 매핑 변경 | 키를 클릭하여 매핑할 키보드 키를 변경합니다.                   |
| 키 위치 조정 | 키를 드래그하여 원하는 위치로 이동합니다.                      |
| 키 크기 조정 | 키를 선택한 후 핸들을 드래그하여 크기를 조정합니다.            |

### Settings 탭

설정 탭에서는 다음 항목을 설정할 수 있습니다:

- **일반 설정**: 언어, 배경색, 렌더링 옵션 등
- **노트 이펙트**: 노트 효과 색상, 속도, 높이 등
- **키 카운터**: 카운터 표시 여부 및 스타일
- **커스텀 CSS/JS**: 사용자 정의 스타일 및 플러그인

## 오버레이 창

오버레이 창은 실제로 키 입력이 시각화되는 투명한 창입니다.

### 특징

- **투명 배경**: 배경이 투명하여 게임 화면 위에 오버레이로 배치할 수 있습니다.
- **클릭 통과**: 설정에서 "클릭 통과" 옵션을 활성화하면 마우스 클릭이 오버레이를 통과합니다.
- **항상 위**: "항상 위에 표시" 옵션으로 다른 창 위에 고정할 수 있습니다.

### OBS에서 사용하기

OBS Studio에서 투명 배경을 유지하려면:

1. **윈도우 캡쳐**를 사용합니다.
2. 캡쳐 방법을 **Windows 10 (1903 이상)**으로 설정합니다.
3. **클라이언트 영역 캡쳐**를 체크합니다.

<Callout type="info">크로마키 없이도 투명 배경이 유지됩니다.</Callout>
