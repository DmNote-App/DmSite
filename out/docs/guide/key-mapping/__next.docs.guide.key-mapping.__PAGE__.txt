# 키 매핑

키 매핑은 키뷰어의 핵심 기능입니다. 각 키에 어떤 키보드 키를 할당할지 설정할 수 있습니다.

## 키 모드 선택

상단의 드롭다운 메뉴에서 키 모드를 선택합니다:

- **4key**: 4개의 키 (리듬게임 기본)
- **5key**: 5개의 키
- **6key**: 6개의 키
- **8key**: 8개의 키

각 모드는 독립적인 설정을 가지며, 모드를 전환해도 이전 설정이 유지됩니다.

## 키 추가하기

1. 그리드의 빈 공간을 **우클릭**합니다.
2. 메뉴에서 **키 추가**를 선택합니다.
3. 새로 생성된 키를 클릭하여 매핑할 키보드 키를 입력합니다.

## 키 삭제하기

1. 삭제할 키를 **우클릭**합니다.
2. 메뉴에서 **키 삭제**를 선택합니다.

## 키 매핑 변경하기

1. 변경할 키를 **클릭**합니다.
2. 모달이 나타나면 새로 매핑할 **키보드 키를 누릅니다**.
3. 매핑이 완료되면 모달이 자동으로 닫힙니다.

<Callout type="info">
  대부분의 키보드 키를 매핑할 수 있습니다. 특수 키(Shift, Ctrl, Alt 등)도
  지원됩니다.
</Callout>

## 키 위치 조정

키를 **드래그**하여 원하는 위치로 이동합니다. 키는 그리드에 맞춰 정렬됩니다.

## 키 크기 조정

1. 크기를 조정할 키를 **클릭**하여 선택합니다.
2. 키 주변에 나타나는 **핸들**을 드래그하여 크기를 조정합니다.

## 키 스타일 설정

각 키에 클래스명을 할당하여 커스텀 CSS로 개별 스타일을 적용할 수 있습니다.

1. 스타일을 변경할 키를 **우클릭**합니다.
2. 메뉴에서 **클래스 설정**을 선택합니다.
3. 클래스명을 입력합니다. (예: `blue`, `special`)

<Callout type="warning">
  클래스명 입력 시 선택자는 제외하고 이름만 입력하세요. - ✅ 올바름: `blue` - ❌
  잘못됨: `.blue`
</Callout>

## 키 이미지 설정

각 키에 이미지를 할당할 수 있습니다.

1. 이미지를 설정할 키를 **우클릭**합니다.
2. 메뉴에서 **이미지 설정**을 선택합니다.
3. 이미지 파일을 선택합니다.
