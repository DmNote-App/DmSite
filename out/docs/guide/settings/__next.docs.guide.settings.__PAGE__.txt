# 설정

DM Note의 다양한 설정 옵션을 안내합니다.

## 일반 설정

### 언어

프로그램 인터페이스의 언어를 선택합니다:

- **한국어**
- **English**

### 배경색

오버레이 창의 배경색을 설정합니다. 기본값은 투명입니다.

<Callout type="info">
  OBS에서 윈도우 캡쳐를 사용하면 투명 배경이 유지됩니다. 크로마키가 필요
  없습니다.
</Callout>

## 창 설정

### 항상 위에 표시

오버레이 창이 다른 창 위에 항상 표시되도록 합니다. 게임 화면 위에 키뷰어를 배치할 때 유용합니다.

### 오버레이 창 고정

오버레이 창의 위치를 고정하여 실수로 이동하는 것을 방지합니다.

### 클릭 통과

활성화하면 오버레이 창을 통과하여 아래 창을 클릭할 수 있습니다.

<Callout type="warning">
  클릭 통과를 활성화하면 오버레이 창을 직접 조작할 수 없게 됩니다. 메인 창에서
  설정을 변경하세요.
</Callout>

## 그래픽 설정

### 렌더링 옵션

그래픽 렌더링 방식을 선택합니다:

- **자동**: 시스템에 맞는 최적의 옵션을 자동 선택
- **Direct3D 11**: 대부분의 최신 시스템에 권장
- **Direct3D 9**: 구형 시스템 호환용
- **OpenGL**: DirectX에 문제가 있는 경우 대안

<Callout type="info">
  그래픽 문제가 발생하면 렌더링 옵션을 변경해 보세요.
</Callout>

### 리사이즈 기준점

오버레이 창 크기 조절 시 기준이 되는 지점을 선택합니다:

- **좌상단**: 좌상단 고정, 우하단으로 확장
- **중앙**: 중앙 고정, 모든 방향으로 확장
- **기타 옵션**: 다양한 기준점 선택 가능

## 커스텀 CSS/JS

### 커스텀 CSS

사용자 정의 CSS 파일을 로드하여 키와 카운터의 스타일을 커스터마이징합니다.

1. **CSS 파일 선택** 버튼을 클릭합니다.
2. CSS 파일을 선택합니다.
3. 스타일이 즉시 적용됩니다.

<Callout type="info">
  CSS 스타일링에 대한 자세한 내용은 [커스텀 CSS 가이드](/custom-css)를
  참조하세요.
</Callout>

### JS 플러그인

JavaScript 플러그인을 로드하여 프로그램 기능을 확장합니다.

1. **JS 플러그인 활성화** 토글을 켭니다.
2. **플러그인 관리** 버튼을 클릭합니다.
3. **JS 플러그인 추가** 버튼으로 파일을 선택합니다.

<Callout type="info">
  플러그인 개발에 대한 자세한 내용은 [플러그인 시스템](/getting-started)을
  참조하세요.
</Callout>

## 설정 초기화

모든 설정을 기본값으로 되돌리려면:

1. **설정 초기화** 버튼을 클릭합니다.
2. 확인 메시지에서 **확인**을 선택합니다.

<Callout type="warning">
  설정 초기화는 되돌릴 수 없습니다. 중요한 설정은 미리 프리셋으로 저장해 두세요.
